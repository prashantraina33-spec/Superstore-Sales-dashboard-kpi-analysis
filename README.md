# Superstore Sales Dashboard & KPI Analysis

## 📊 Project Overview
This project analyzes sales data using Python and builds an interactive dashboard in Excel.

## 🔧 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Excel (Pivot Tables, KPI Cards, Dashboard)

## 📈 Key Insights
- High sales ≠ high profit
- Few products generate most profit (Pareto effect)
- Regional performance varies
- Seasonal trends observed

## 🛠️ Process
1. Data Cleaning (handled missing values, data types)
2. Feature Engineering (Total Sales, Profit)
3. Outlier Removal (IQR method)
4. Data Visualization
5. Dashboard Creation in Excel

## ✅ Conclusion
This project demonstrates that data storytelling goes 
beyond raw numbers. Key takeaways include:

- High sales do not always translate to high profit
- A few products drive the majority of revenue 
  (Pareto Principle)
- Regional and seasonal trends are critical for 
  strategic planning
- Excel dashboards combined with Python analysis 
  provide powerful, actionable business insights

This end-to-end analysis strengthens decision-making 
by turning raw sales data into meaningful KPIs 
and visual insights.
